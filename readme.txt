Plugin umożliwia wyświetlenie kalendarza dietetyka z platformy dietetyk.pl za pomocą shortcode

== Description ==

Wtyczka służy do instalacji kalendarza online z portalu dietetyk.pl. Dzięki jej zainstalowaniu każdy dietetyk będzie mógł na swojej stronie umieścić kalendarz do umawiania wizyt. Dzięki temu odwiedzający stronę w szybki i prosty sposób będą mogli zapisać się na spotkanie z dietetykiem w dogodnym dla nich terminie.

Żeby korzystać z wtyczki należy posiadać konto dietetyka na portalu Dietetyk.pl oraz aktywować kalendarz online. W celu aktywacji kalendarza należy ustawić dni oraz godziny pracujące, dodać zakres oferowanych usług oraz potwierdzić swój nr telefonu.

Aby dodać kalendarz dietetyka umieść jego alias pomiędzy nawiasami przy src w shortcode: [dpc_dietetyk src=""]. Alias jest końcowką adresu na platformie https://dietetyk.pl. Przykład: [dpc_dietetyk src="adam.kowalski"]

== Installation ==

1. Pobierz plugin i umieść go w katalogu `/wp-content/plugins/plugin-name` bądź zainstaluj go poprzez instalator wtyczek Wordpressa.
2. Aktywuj plugin na ekranie 'Plugins' w Wordpresss
3. Umieść shortcode na Twojej stronie by wyświetlić kalendarz [dpc_dietetyk src="adam.kowalski"]


== Frequently Asked Questions ==

= Jak założyć konto na dietetyku =

Wejdź na https://dietetyk.pl, następnie wybierz z menu: "Jesteś dietetykiem?", kliknij guzik "Załóż darmowe konto"
